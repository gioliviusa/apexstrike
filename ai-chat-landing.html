Deployment Steps

  1. Create a GitHub repository named apexstryke.github.io
  2. Upload this HTML file as index.html to the main branch
  3. Go to repository Settings → Pages → Select source branch
  4. Set custom domain to apexstryke.com in GitHub Pages settings
  5. Configure DNS settings with your domain provider to point to GitHub Pages
  6. Wait for DNS propagation (up to 24 hours)

Troubleshooting Tips

  • DNS Propagation - Changes may take up to 24 hours to take effect
  • Custom Domain - Verify domain is set in GitHub Pages settings
  • HTTPS - GitHub Pages provides automatic HTTPS for custom domains
  • Repository Name - Must be exactly username.github.io
  • Test Deployment:
    curl -I https://apexstryke.com
